# gulp-starter

This is a Gulp4 starter. 

## Getting started
You will need:
- Node
  - [Download Node here](https://nodejs.org/download/)
- Gulp
  - Run `npm i -g gulp`

##### 1. Clone this repo
```bash
git clone https://github.com/jeffersonlam/gulp-starter.git
```
##### 2. In the cloned folder, install the dependencies
```bash
cd gulp-starter
npm install
```
##### 3. Start gulp
```bash
gulp
```
##### 4. Write some code!
